Improvised Gadget

Requirements

  • Gadgeteer 1
  • Improvised Gadget talent

Action Type

Action (Cost: 2 energy)

Description

You rapidly assemble a functional device from spare parts, half-finished components, and pure ingenuity.

Effect

You create one Level 0 gadget that you did not prepare. You can only use this gadget once.
